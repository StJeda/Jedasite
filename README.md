# Jedasite

## О проекте  

Jedasite — это веб-платформа, разработанная для эффективной интеграции с украинскими клиентами и международной аудиторией. Проект направлен на обеспечение удобного взаимодействия и упрощение бизнес-процессов для пользователей по всему миру.  

## Основной функционал  

- Микросервисная архитектура для гибкости и масштабируемости.  
- Поддержка серверного рендеринга для повышения производительности и SEO-оптимизации.  
- Асинхронная обработка задач через брокер сообщений (RabbitMQ).  

## Стек технологий  

### **Фронтенд**  
- **Next.js**: мощный React-фреймворк с поддержкой серверного рендеринга (SSR).  

### **Бэкенд**  
- **ASP.NET Core Web API**: современная платформа для построения высокопроизводительных RESTful API.  
- **Minimal API**: облегчённый подход к созданию API для повышения скорости разработки.  
- **RabbitMQ**: брокер сообщений для реализации асинхронной коммуникации между микросервисами.
